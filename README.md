> 🌐 **Canlı Yayın Adresi:** Bu proje aktif olarak **[aliguvenuygulamaoteli.com](https://aliguvenuygulamaoteli.com/)** adresinde yayındadır.

React ve PHP + MySQL kullanılarak geliştirilen bu full-stack projede, otel işletmesi için modern bir web sitesi ve tüm dinamik içeriklerin (oda, hizmet, görsel) teknik bilgi gerektirmeksizin yönetilebildiği kapsamlı bir admin paneli (CMS) inşa edilmiştir.

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Admin Panel:** React
* **Backend / API:** PHP
* **Veritabanı:** MySQL
